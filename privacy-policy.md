# Privacy Policy — ProdTrack ToolWeb

ProdTrack ToolWeb does not collect, store, or transmit any personal data.

The extension only accesses the active browser tab when the user explicitly clicks a script in the popup. No data from visited pages is read, stored, or sent anywhere.

A list of available scripts is fetched from a GitHub repository and cached locally in the browser for performance. This cache contains only script names and code — no user data.
